# Spotify-Control                                 

## Shortcuts

CTRL + ALT + SPACE  Play/Pause
CTRL + ALT + L      Next Track
CTRL + ALT + J      Previous Track
CTRL + ALT + B      Browse Library Tracks
CTRL + ALT + R      Select Repeat Mode
CTRL + ALT + S      Toggle Shuffle

## Status Bar

"sublimify_status" : false

to disable status bar info.
